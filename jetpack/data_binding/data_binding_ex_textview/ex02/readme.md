# 시작하며...

이 샘플 프로젝트는 데이터 바인딩을 텍스트 뷰에 적용하는 2번째 예를 보여 드립니다.

---

# 스크린샷

MainActivity에 1개의 텍스트 뷰가 있고, 이것은 앱 시작 후 경과된 시간을 시, 분, 초로 보여줍니다. 이 뷰는 hour, minute, second 변수들와 바인딩 되어 있습니다. 이 변수들의 값을 바꾸면 텍스트 뷰의 내용이 바뀌는 것이지요.
